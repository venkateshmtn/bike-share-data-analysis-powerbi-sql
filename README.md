# 🚴 Bike Share Data Analysis Dashboard

## 📌 Project Overview

This project presents an end-to-end **data analysis and visualization solution** for a bike-sharing dataset. The dashboard provides actionable insights into user behavior, revenue trends, and operational performance using interactive visuals.

The analysis is built using multiple CSV datasets, processed in SQL, and visualized in Power BI for clear and dynamic reporting.

---

## 📂 Dataset Description

The project uses three datasets:

* **bike_share_yr_0.csv** – Historical bike-sharing data (Year 0)
* **bike_share_yr_1.csv** – Historical bike-sharing data (Year 1)
* **cost_table.csv** – Cost and pricing-related information

These datasets were combined and transformed to create meaningful insights.

---

## 🛠️ Tools & Technologies Used

* **MS SQL Server** – Data cleaning, transformation, and querying
* **Power BI** – Dashboard creation and data visualization
* **CSV Files** – Raw data source

---

## ⚙️ Data Processing Workflow

1. Imported CSV datasets into SQL Server
2. Performed data cleaning and handled missing values
3. Joined multiple tables to create a unified dataset
4. Created calculated fields for revenue and trends
5. Loaded processed data into Power BI
6. Built interactive dashboards and reports

---

## 📊 Dashboard Features

* 📈 Year-wise revenue and usage trends
* 🚴 Bike usage analysis by season and time
* 👥 Customer segmentation (casual vs registered users)
* 💰 Cost and profit insights
* 📅 Time-based analysis (daily, monthly patterns)
